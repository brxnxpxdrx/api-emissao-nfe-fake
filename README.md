# 🧾 Fake NF-e API / PDV

Uma API simulada para **emissão, consulta, inutilização e cancelamento de notas fiscais**, ideal para testes de PDV sem integração real com a SEFAZ.

---

## 🛠 Stack
- ⚡ Node.js  
- 💙 TypeScript  
- 🌐 Express  
- 📝 JSON como armazenamento local em memória
